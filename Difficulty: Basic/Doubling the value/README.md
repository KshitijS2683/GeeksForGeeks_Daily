<h2><a href="https://www.geeksforgeeks.org/problems/doubling-the-value4859/1?page=7&difficulty=Basic&sortBy=submissions">Doubling the value</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an array <strong>arr</strong> and an integer <strong>b</strong>, traverse the array (from the beginning) and if the element in array is b, <strong>double</strong> b and continue traversal. </span><span style="font-size: 18px;">Find the value of b after the complete traversal.</span></p>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> b = 2, arr[] = [1, 2, 3, 4, 8]
<strong>Output:</strong> 16
<strong>Explanation</strong>: b is initially 2. We get 2 at the 1st index, hence b becomes 4. <br>Next, we get b 3rd time, hence b becomes 8. <br>Next, we get b 4th time, hence b becomes 16.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>b = 3, arr[] = [1, 2, 3, 4, 8]
<strong>Output:</strong> 6
<strong>Explanation</strong>: b is initially 3. We get 3 2nd time, hence b becomes 6.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong><br>1 ≤ arr.size() ≤ 20<br>1 ≤ b ≤ 10<sup>3</sup><br>1 ≤ arr[i] ≤ 10<sup>9</sup></span></p></div><p><span style=font-size:18px><strong>Company Tags : </strong><br><code>Flipkart</code>&nbsp;<br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Searching</code>&nbsp;<code>Algorithms</code>&nbsp;
